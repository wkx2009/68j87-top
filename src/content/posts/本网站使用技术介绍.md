## 🧱 核心框架

| 项目 | 说明 |
|:---|:---|
| **[Astro](https://astro.build)** | 现代化静态站点生成器，以群岛架构为核心，默认静态输出，极简 JavaScript |
| **[Fuwari](https://github.com/saicaca/fuwari)** | 基于 Astro 与 Tailwind CSS 的博客主题，性能优异，设计简洁 |
| **[pnpm](https://pnpm.io)** | Node.js 包管理器 |
| **[Tailwind CSS](https://tailwindcss.com)** | CSS 框架，内置暗色模式与大量预设 |

## ✍️ 内容与排版

| 技术 | 说明 |
|:---|:---|
| **[Markdown / MDX](https://markdown.com.cn)** | 文章创作格式 |
| **[Expressive Code](https://expressive-code.com)** | 代码块语法高亮引擎 |
| **[KaTeX](https://katex.org)** | LaTeX数学公式渲染 |
| **[remark / rehype](https://github.com/remarkjs)** | Markdown 处理插件管线 |
| **[Pagefind](https://pagefind.app)** | 全文搜索引擎 |

## 🖼️ 图片与资源

| 技术 | 说明 |
|:---|:---|
| **[Sharp](https://sharp.pixelplumbing.com/install)** | 高性能 Node.js 图像处理库 |
| **[PhotoSwipe](https://photoswipe.com)** | 图片灯箱预览，支持手势与缩放 |
| **[OverlayScrollbars](https://kingsora.github.io/OverlayScrollbars)** | 自定义滚动条样式 |
| **[Iconify](https://iconify.design)** | 统一图标框架 |

## ✒️ 字体与排版

| 技术 | 说明 |
|:---|:---|
| **[MiSans](https://hyperos.mi.com/font/zh/download)** | 小米官方无衬线字体，免费可商用，本站中文主字体 |
| **[misans-webfont](https://www.npmjs.com/package/misans-webfont)** | MiSans 字体分包方案，按需加载，大幅减少传输体积 |
## ☁️ 部署与性能
